# Just a simple HTML file can be enough sometimes!

**Follow this to fully enjoy the APP


1. Download the Raw .HTML file and open it on your browser
2. Export a .TSV on the button inside the .HTML
3. Use it's content as a mold for any A.I. of your liking to build a list of words of your chosing, pick up a theme, set a limit for words
4. Save another .TSV with those words and import via the button inside the .HTML, It will recognize and build new flashcards
5. 努力學習啊！
